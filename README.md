# Jarvis 1.0

A personal finance tracker app built with Flutter.

## Features

- **Transaction Management**: Add, edit, and delete financial transactions
- **User Management**: Track finances for multiple users
- **Bill Splitting**: Split bills equally among selected users
- **Transaction Log**: Automatic logging of all transactions to a text file
- **Export**: Export data as JSON or CSV
- **Summary View**: See totals owed per user

## Getting Started

1. Install dependencies:
   ```bash
   flutter pub get
   ```

2. Run the app:
   ```bash
   flutter run
   ```

## App Structure

- **Main Screen**: Finances management interface
- **Data Storage**: Transactions and users stored in JSON files
- **Transaction Log**: All transactions logged to `transactions_log.txt`
- **Export**: Share data as JSON or CSV files
